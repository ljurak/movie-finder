## General info

Movie Finder allows you to find info about your favourite movie.

## How to start

```bash
git clone https://github.com/ljurak/movie-finder.git
cd movie-finder
npm install
npm start
```
The app will start on http://localhost:3000

## Technologies

Project is created with:
* JavaScript
* ReactJS
* HTML5
* CSS3
